## 二、模块详情

### 2.1 用户系统 (User Service)

| 功能 | 说明 |
|------|------|
| 注册/登录 | 手机号验证码、密码、微信/QQ 第三方登录 |
| 个人资料 | 昵称、头像、简介、所在城市、养宠年限 |
| 宠物档案 | **多宠物管理**：品种、年龄、体重、性别、疫苗记录、绝育状态、性格标签 |
| 关注系统 | 关注/粉丝、关注列表、互关判定 |
| 隐私设置 | 动态可见范围、位置可见、消息屏蔽 |

### 2.2 内容社区 (Feed + Social Service)

| 功能 | 说明 |
|------|------|
| 图文动态 | 发布照片(最多9张) + 文字描述 + 定位 + 话题标签 |
| 短视频 | 15-60秒短视频、配乐、滤镜 |
| 互动 | 点赞 ❤️、评论 💬、收藏 ⭐、转发 🔄 |
| 信息流 | 关注流、推荐流、同城流、热门流 |
| 话题/圈子 | #话题标签、宠物品种圈、地区圈、兴趣圈 |
| 同城匹配 | LBS 定位，推荐附近宠物/用户 |
| 约遛/聚会 | 发起约遛 → 时间/地点/人数 → 参与者确认 |

### 2.3 交易市场 (Marketplace Service)

#### 活体宠物交易

| 流程 | 说明 |
|------|------|
| 卖家发布 | 品种/年龄/疫苗/价格/视频 + 资质审核 |
| 买家浏览 | 按品种/地区/价格筛选，查看详情 |
| 担保交易 | 买家付款至平台 → 线下交接确认 → 平台放款 |
| 信用体系 | 卖家实名认证 + 历史交易评价 |
| 合规审核 | 禁售品种过滤、繁殖许可证校验 |

#### 宠物用品商城

| 功能 | 说明 |
|------|------|
| 商品管理 | SKU / 库存 / 价格 / 规格 / 多图 |
| 购物车 | 增删改、批量结算 |
| 订单系统 | 待付款/待发货/待收货/已完成/退款售后 |
| 商家入驻 | 资质审核、店铺装修、数据看板 |
| 物流对接 | 快递 100 等物流查询 |

#### 二手闲置

- C2C 二手宠物用品交易
- 一键转卖、价格建议

### 2.4 服务预约 (Service Booking)

| 服务类型 | 功能 |
|---------|------|
| 美容/Grooming | 选择店铺 → 选择服务项目 → 选择时间段 → 在线支付 → 到店消费 |
| 医疗服务 | 附近医院 → 科室/医生 → 预约挂号 → 在线问诊(可选) |
| 寄养/日托 | 查看环境/评价 → 预约日期 → 实时反馈(照片/视频) |
| 训练服务 | 课程选择 → 一对一/小班 → 训练场地 |
| 评价体系 | 服务后评价(星级 + 文字 + 图片) |

### 2.5 即时通讯 (IM Service)

- 私信聊天（文本 + 图片 + 语音）
- 已读/未读
- 消息推送（App Push + WebSocket）
- 敏感词过滤
- 交易/服务咨询专用会话

### 2.6 搜索与推荐

| 功能 | 技术方案 |
|------|---------|
| 全文搜索 | Elasticsearch：宠物/商品/用户/动态 |
| 搜索联想 | 前缀匹配 + 热词推荐 |
| 信息流推荐 | v1: 规则推荐 | v2: 协同过滤 + 向量召回 |
| 商品推荐 | 基于浏览/购买历史 + 宠物品种匹配 |
| 同城推荐 | GeoHash + Redis GEO |

### 2.7 管理后台

| 模块 | 功能 |
|------|------|
| 内容审核 | 动态/评论图片视频人工审核、违规处理 |
| 用户管理 | 封禁/解封、身份认证审核 |
| 交易监管 | 活体交易审核、纠纷仲裁、退款处理 |
| 商家管理 | 入驻审核、店铺管理、结算管理 |
| 数据统计 | DAU/MAU、交易额、热门品类、用户画像 |
| 运营工具 | 内容置顶、Push 推送、活动创建、积分发放 |
